Random A-Frame Entity Generator Component Demos

Experimental A-Frame build to explore creating generative randomness.

Desc: On load generate a randomised 3D sculpture with random primitives, scale, textures, colors and random positioning based on seed values Assign random textures if texture mode is on (Optional), random colour to each shape Two main modes - Load random primitives or load random models (Optional) in glb format (in Schema textures / custumGLB - Boolean) /Seed values that can be modified to mod amount of randomness / spread and scale Min / Max values to determine amount of minimum / Maximum shapes to generate Random lights generation (Optional)
